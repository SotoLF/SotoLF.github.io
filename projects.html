Epitope Evaluator

Interactive web application for T-cell epitope prediction and evaluation

Epitope Evaluator is a comprehensive web tool designed to facilitate vaccine development by providing accurate predictions of T-cell epitopes. The tool integrates multiple prediction algorithms, population coverage analysis, and interactive visualization to help researchers identify promising vaccine candidates.

The software was developed to address the need for more accessible and user-friendly tools in immunoinformatics, particularly for researchers working on vaccine design against emerging pathogens. It has been successfully applied in COVID-19 vaccine research and other infectious disease studies.

Key Features

MHC binding affinity prediction using multiple algorithms
Population coverage analysis for diverse ethnic groups
Immunogenicity assessment based on peptide properties
Epitope conservation analysis across strains
Interactive visualization of prediction results
Batch processing of protein sequences

Technologies Used

Python Flask React D3.js Scikit-learn Pandas Biopython Docker

TF-NetViz

Visualization tool for transcription factor regulatory networks

TF-NetViz is a specialized tool for visualizing and analyzing transcription factor regulatory networks. The tool enables researchers to understand complex gene regulation patterns, identify key regulatory hubs, and explore the dynamics of transcription factor interactions in different biological contexts.

Developed as part of my research on transcription factor effector domains, this tool addresses the challenge of interpreting large-scale protein-DNA interaction data. It provides intuitive visualizations and analysis features that help researchers gain insights into the complex mechanisms of gene regulation.

Key Features

Interactive network visualization with customizable layouts
Integration of multiple data types (ChIP-seq, ATAC-seq, expression)
Network motif detection and analysis
Differential network analysis between conditions
Network centrality metrics and hub identification
Export options for publication-quality figures

Technologies Used

R Shiny D3.js NetworkX Bioconductor igraph ggplot2

COVID Biomarker Pipeline

Analysis pipeline for identifying and validating COVID-19 biomarkers

The COVID Biomarker Pipeline is a comprehensive computational framework designed to identify and validate potential biomarkers for COVID-19 from clinical data. The pipeline integrates various data types, including clinical measurements, immunological parameters, and genomic data, to identify markers associated with disease severity and progression.

Developed in response to the COVID-19 pandemic, this tool has been used to identify several novel biomarkers, including the leukocyte glucose index, which was published as a predictive marker for COVID-19 severity. The pipeline includes machine learning models for biomarker validation and risk prediction.

Key Features

Multi-omics data integration framework
Feature selection algorithms for biomarker discovery
Machine learning models for outcome prediction
Statistical validation of potential biomarkers
Visualization of biomarker performance metrics
Scalable processing of large clinical datasets

Technologies Used

Python Scikit-learn Pandas Numpy Matplotlib Seaborn Statsmodels Snakemake
